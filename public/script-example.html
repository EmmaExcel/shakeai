Shake Cursor Example

This page demonstrates how to load the Shake Cursor SDK using a simple <script> tag.

To test the AI features:

  1. Shake your mouse rapidly left and right.
  2. Or press Cmd + K (Mac) / Ctrl + K (Windows).
  3. Select any text on this page to ask a question.

Retrieval Augmented Generation (RAG) is a technique that gives large language models access to specific, external data to improve the accuracy and relevance of their responses.

By using Shake Cursor, site owners can index their documentation and provide instant, context-aware help to their visitors without forcing them to leave the current page.
